# hack-your-education-ece

## Show slides

Prerequisites:

Install node from [https://nodejs.org/en/](https://nodejs.org/en/)

### Install NPM packages

1. Open Powershell/Terminal/CMD
2. Navigate to `./slides` folder
3. Run `npm install`

### Serve slides

1. Open Powershell/Terminal/CMD
2. Navigate to `./slides` folder
3. Run `npm start`
4. Open browser an go to `localhost:8000`
